The normal equations

We now solve the general linear least-squares problem in Definition 3.1.1. That is, given A∈Rm×n\mathbf{A}\in\mathbb{R}^{m \times n}A∈Rm×n and b∈Rm\mathbf{b}\in\mathbb{R}^mb∈Rm, with m>nm>nm>n, find the x∈Rn\mathbf{x}\in\mathbb{R}^nx∈Rn that minimizes ∥b−Ax∥2\| \mathbf{b} - \mathbf{A}\mathbf{x} \|_2∥b−Ax∥2​.

There is a concise explicit solution. In the following proof we make use of the elementary algebraic fact that for two vectors u\mathbf{u}u and v\mathbf{v}v,

(u+v)T(u+v)=uTu+uTv+vTu+vTv=uTu+2vTu+vTv. (\mathbf{u}+\mathbf{v})^T(\mathbf{u}+\mathbf{v}) = \mathbf{u}^T\mathbf{u} + \mathbf{u}^T\mathbf{v} + \mathbf{v}^T\mathbf{u} + \mathbf{v}^T\mathbf{v} = \mathbf{u}^T\mathbf{u} + 2\mathbf{v}^T\mathbf{u} + \mathbf{v}^T\mathbf{v}.(u+v)T(u+v)=uTu+uTv+vTu+vTv=uTu+2vTu+vTv.
(3.2.1)
Theorem 3.2.1

If x\mathbf{x}x satisfies AT(Ax−b)=0\mathbf{A}^T(\mathbf{A}\mathbf{x}-\mathbf{b})=\boldsymbol{0}AT(Ax−b)=0, then x\mathbf{x}x solves the linear least-squares problem, i.e., x\mathbf{x}x minimizes ∥b−Ax∥2\| \mathbf{b}-\mathbf{A}\mathbf{x} \|_2∥b−Ax∥2​.

Proof

Let y∈Rn\mathbf{y}\in \mathbb{R}^ny∈Rn be any vector. Then A(x+y)−b=Ax−b+Ay\mathbf{A}(\mathbf{x}+\mathbf{y})-\mathbf{b}=\mathbf{A}\mathbf{x}-\mathbf{b}+\mathbf{A}\mathbf{y}A(x+y)−b=Ax−b+Ay, and

∥A(x+y)−b∥22=[(Ax−b)+(Ay)]T[(Ax−b)+(Ay)]=(Ax−b)T(Ax−b)+2(Ay)T(Ax−b)+(Ay)T(Ay)=∥Ax−b∥22+2yTAT(Ax−b)+∥Ay∥22=∥Ax−b∥22+∥Ay∥22≥∥Ax−b∥22.\begin{split} \| \mathbf{A}(\mathbf{x}+\mathbf{y})-\mathbf{b} \|_2^2 &= [(\mathbf{A}\mathbf{x}-\mathbf{b})+(\mathbf{A}\mathbf{y})]^T[(\mathbf{A}\mathbf{x}-\mathbf{b})+(\mathbf{A}\mathbf{y})]\\ &= (\mathbf{A}\mathbf{x}-\mathbf{b})^T(\mathbf{A}\mathbf{x}-\mathbf{b}) + 2(\mathbf{A}\mathbf{y})^T(\mathbf{A}\mathbf{x}-\mathbf{b}) + (\mathbf{A}\mathbf{y})^T(\mathbf{A}\mathbf{y})\\ &= \| \mathbf{A}\mathbf{x}-\mathbf{b} \|_2^2 + 2\mathbf{y}^T \mathbf{A}^T(\mathbf{A}\mathbf{x}-\mathbf{b}) + \| \mathbf{A}\mathbf{y} \|_2^2 \\ &= \| \mathbf{A}\mathbf{x}-\mathbf{b} \|_2^2 + \| \mathbf{A}\mathbf{y} \|_2^2 \\ & \ge \| \mathbf{A}\mathbf{x}-\mathbf{b} \|_2^2. \end{split}∥A(x+y)−b∥22​​=[(Ax−b)+(Ay)]T[(Ax−b)+(Ay)]=(Ax−b)T(Ax−b)+2(Ay)T(Ax−b)+(Ay)T(Ay)=∥Ax−b∥22​+2yTAT(Ax−b)+∥Ay∥22​=∥Ax−b∥22​+∥Ay∥22​≥∥Ax−b∥22​.​
(3.2.2)
Definition 3.2.1 (Normal equations)

Given A∈Rm×n\mathbf{A}\in \real^{m\times n}A∈Rm×n and b∈Rm\mathbf{b}\in \real^{m}b∈Rm, the normal equations for the linear least-squares problem argmin⁡∥b−Ax∥\operatorname{argmin}\| \mathbf{b}- \mathbf{A} \mathbf{x}\|argmin∥b−Ax∥ are AT(Ax−b)=0\mathbf{A}^T(\mathbf{A}\mathbf{x}-\mathbf{b})=\boldsymbol{0}AT(Ax−b)=0, or equivalently,

ATAx=ATb.\mathbf{A}^T\mathbf{A}\mathbf{x}=\mathbf{A}^T\mathbf{b}.ATAx=ATb.
(3.2.3)

The normal equations have a geometric interpretation, as shown in Figure 3.2.1. The vector in the range (column space) of A\mathbf{A}A that lies closest to b\mathbf{b}b makes the vector difference Ax−b\mathbf{A}\mathbf{x}-\mathbf{b}Ax−b perpendicular to the range. Thus for any z\mathbf{z}z, we must have (Az)T(Ax−b)=0(\mathbf{A} \mathbf{z})^T(\mathbf{A}\mathbf{x}-\mathbf{b})=0(Az)T(Ax−b)=0, which is satisfied if AT(Ax−b)=0\mathbf{A}^T(\mathbf{A}\mathbf{x}-\mathbf{b})=\boldsymbol{0}AT(Ax−b)=0.

3.2.1Pseudoinverse and definiteness¶

If we associate the left-hand side of the normal equations as (ATA) x(\mathbf{A}^T\mathbf{A})\,\mathbf{x}(ATA)x, we recognize (3.2.3) as a square n×nn\times nn×n linear system to solve for x\mathbf{x}x.

Definition 3.2.2 (Pseudoinverse)

If A∈Rm×n\mathbf{A}\in\real^{m\times n}A∈Rm×n with m>nm>nm>n, its pseudoinverse is the n×mn\times mn×m matrix

A+=(ATA)−1AT.\mathbf{A}^+ = (\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T.A+=(ATA)−1AT.
(3.2.4)

Mathematically, the overdetermined least-squares problem Ax≈b\mathbf{A}\mathbf{x}\approx \mathbf{b}Ax≈b has the solution x=A+b\mathbf{x}=\mathbf{A}^+\mathbf{b}x=A+b.

Computationally we can generalize the observation about Julia from Chapter 2: backslash is equivalent mathematically to left-multiplication by the inverse (in the square case) or pseudoinverse (in the rectangular case) of a matrix. One can also compute the pseudoinverse directly using pinv, but as with matrix inverses, this is rarely necessary in practice.

The matrix ATA\mathbf{A}^T\mathbf{A}ATA appearing in the pseudoinverse has some important properties.

Theorem 3.2.2

For any real m×nm\times nm×n matrix A\mathbf{A}A with m≥nm\ge nm≥n, the following are true:

  1. ATA\mathbf{A}^T\mathbf{A}ATA is symmetric.

  2. ATA\mathbf{A}^T \mathbf{A}ATA is singular if and only if the columns of A\mathbf{A}A are linearly dependent. (Equivalently, if and only if the rank of A\mathbf{A}A is less than nnn.)

  3. If ATA\mathbf{A}^T\mathbf{A}ATA is nonsingular, then it is positive definite.

Proof

The first part is left as Exercise 3.2.3. For the second part, suppose that ATAz=0\mathbf{A}^T\mathbf{A}\mathbf{z}=\boldsymbol{0}ATAz=0. Note that ATA\mathbf{A}^T\mathbf{A}ATA is singular if and only if z\mathbf{z}z may be nonzero. Left-multiplying by zT\mathbf{z}^TzT, we find that

0=zTATAz=(Az)T(Az)=∥Az∥22,0 = \mathbf{z}^T\mathbf{A}^T\mathbf{A}\mathbf{z}=(\mathbf{A}\mathbf{z})^T(\mathbf{A}\mathbf{z}) = \| \mathbf{A}\mathbf{z} \|_2^2,0=zTATAz=(Az)T(Az)=∥Az∥22​,
(3.2.5)

which is equivalent to Az=0\mathbf{A}\mathbf{z}=\boldsymbol{0}Az=0. Then z\mathbf{z}z may be nonzero if and only if the columns of A\mathbf{A}A are linearly dependent.

Finally, we can repeat the manipulations above to show that for any nonzero nnn-vector v\mathbf{v}v, vT(ATA)v=∥Av∥22≥0\mathbf{v}^T(\mathbf{A}^T\mathbf{A})\mathbf{v}=\| \mathbf{A}\mathbf{v} \|_2^2\ge 0vT(ATA)v=∥Av∥22​≥0, and equality is not possible thanks to the second part of the theorem.

3.2.2Implementation¶

The definition of the pseudoinverse involves taking the inverse of a matrix, so it is not advisable to use the pseudoinverse computationally. Instead, we use the definition of the normal equations to set up a linear system, which we already know how to solve. In summary, the steps for solving the linear least squares problem Ax≈b\mathbf{A}\mathbf{x}\approx\mathbf{b}Ax≈b are:

Algorithm 3.2.1 (Solution of linear least squares by the normal equations)

  1. Compute N=ATA\mathbf{N}=\mathbf{A}^T\mathbf{A}N=ATA.

  2. Compute z=ATb\mathbf{z} = \mathbf{A}^T\mathbf{b}z=ATb.

  3. Solve the n×nn\times nn×n linear system Nx=z\mathbf{N}\mathbf{x} = \mathbf{z}Nx=z for x\mathbf{x}x.

Steps 1 and 3 of Algorithm 3.2.1 dominate the flop count.

In the last step we can exploit the fact, proved in Theorem 3.2.2, that N\mathbf{N}N is symmetric and positive definite, and use Cholesky factorization as in Exploiting matrix structure. This detail is included in Function 3.2.2.

"""
    lsnormal(A, b)

Solve a linear least-squares problem by the normal equations.
Returns the minimizer of ||b-Ax||.
"""
function lsnormal(A, b)
    N = A' * A
    z = A' * b
    R = cholesky(N).U
    w = forwardsub(R', z)                   # solve R'z=c
    x = backsub(R, w)                       # solve Rx=z
    return x
end

def lsnormal(A, b):
    """
    lsnormal(A, b)
    
    Solve a linear least squares problem by the normal equations. Returns the
    minimizer of ||b-Ax||.
    """
    N = A.T @ A
    z = A.T @ b
    R = scipy.linalg.cholesky(N)
    w = forwardsub(R.T, z)                   # solve R'z=c
    x = backsub(R, w)                        # solve Rx=z
    return x

3.2.3Conditioning and stability¶

We have already used A\b as the native way to solve the linear least-squares problem Ax≈b\mathbf{A}\mathbf{x}\approx\mathbf{b}Ax≈b in Julia. The algorithm employed by the backslash does not proceed through the normal equations, because of instability.

The conditioning of the linear least-squares problem relates changes in the solution x\mathbf{x}x to those in the data, A\mathbf{A}A and b\mathbf{b}b. A full accounting of the condition number is too messy to present here, but we can get the main idea. We start by generalizing our previous definition of the matrix condition number.

Definition 3.2.3 (Matrix condition number (rectangular case))

If A\mathbf{A}A is m×nm\times nm×n with m>nm > nm>n, then its condition number is defined to be

κ(A)=∥A∥2⋅∥A+∥2.\kappa(\mathbf{A}) = \|\mathbf{A}\|_2 \cdot \|\mathbf{A}^{+}\|_2.κ(A)=∥A∥2​⋅∥A+∥2​.
(3.2.6)

If the rank of A \mathbf{A}A is less than nnn (i.e., if it has linearly dependent columns), then κ(A)=∞\kappa(\mathbf{A})=\inftyκ(A)=∞.

Provided that the minimum residual norm ∥b−Ax∥\|\mathbf{b}-\mathbf{A}\mathbf{x}\|∥b−Ax∥ is relatively small, the conditioning of the linear least-squares problem is close to κ(A)\kappa(\mathbf{A})κ(A).

As an algorithm, the normal equations begin by computing the data for the n×nn\times nn×n system (ATA)x=ATb(\mathbf{A}^T\mathbf{A})\mathbf{x} = \mathbf{A}^T \mathbf{b}(ATA)x=ATb. When these equations are solved, perturbations to the data can be amplified by a factor κ(ATA)\kappa(\mathbf{A}^T\mathbf{A})κ(ATA).

The following can be proved using results in Chapter 7.

Theorem 3.2.4 (Condition number in the normal equations)

If A\mathbf{A}A is m×nm\times nm×n with m>nm > nm>n, then

κ(ATA)=κ(A)2.\kappa(\mathbf{A}^T\mathbf{A}) = \kappa(\mathbf{A})^2.κ(ATA)=κ(A)2.
(3.2.7)

This squaring of the condition number in the normal equations is the cause of instability. If κ(A)\kappa(\mathbf{A})κ(A) is large, the squaring of it can destabilize the normal equations: while the solution of the least-squares problem is sensitive, finding it via the normal equations makes it doubly so.
