The wave equation

Closely related to the advection equation is the wave equation,

utt−c2uxx=0. u_{tt} - c^2 u_{xx} = 0.utt​−c2uxx​=0.
(12.4.1)

This is our first PDE having a second derivative in time. As in the advection equation, u(x,t)=ϕ(x−ct)u(x,t)=\phi(x-ct)u(x,t)=ϕ(x−ct) is a solution of (12.4.1), but now so is u(x,t)=ϕ(x+ct)u(x,t)=\phi(x+c t)u(x,t)=ϕ(x+ct) for any twice-differentiable ϕ (see Exercise 2). Thus, the wave equation supports advection in both directions simultaneously.

We will use x∈[0,1]x \in [0,1]x∈[0,1] and t>0t>0t>0 as the domain. Because uuu has two derivatives in ttt and in xxx, we need two boundary conditions. We will use the Dirichlet conditions

u(0,t)=u(1,t)=0,t≥0,u(0,t) = u(1,t) = 0, \qquad t \ge 0,u(0,t)=u(1,t)=0,t≥0,
(12.4.2)

and two initial conditions,

u(x,0)=f(x),0≤x≤1,ut(x,0)=g(x),0≤x≤1.\begin{split} u(x,0) &= f(x), \qquad 0 \le x \le 1, \\ u_t(x,0) &= g(x), \qquad 0 \le x \le 1. \end{split}u(x,0)ut​(x,0)​=f(x),0≤x≤1,=g(x),0≤x≤1.​
(12.4.3)

One approach is to discretize both the uttu_{tt}utt​ and uxxu_{xx}uxx​ terms using finite differences:

1τ2(Ui,j+1−2Ui,j+Ui,j−1)=c2h2(Ui+1,j−2Ui,j+Ui−1,j).\frac{1}{\tau^2}(U_{i,j+1} - 2U_{i,j} + U_{i,j-1}) = \frac{c^2}{h^2} (U_{i+1,j} - 2U_{i,j} + U_{i-1,j}).τ21​(Ui,j+1​−2Ui,j​+Ui,j−1​)=h2c2​(Ui+1,j​−2Ui,j​+Ui−1,j​).
(12.4.4)

This equation can be rearranged to solve for Ui,j+1U_{i,j+1}Ui,j+1​ in terms of values at time levels jjj and j−1j-1j−1. Rather than pursue this method, however, we will turn to the method of lines.

12.4.1First-order system¶

In order to be compatible with the standard IVP solvers that we have encountered, we must recast (12.4.1) as a first-order system in time. Using our typical methodology, we would define y=uty=u_ty=ut​ and derive

ut=y,yt=c2uxx.\begin{split} u_t &= y, \\ y_t &= c^2 u_{xx}. \end{split}ut​yt​​=y,=c2uxx​.​
(12.4.5)

However, there is another, less obvious option for reducing to a first-order system:

ut=zx,zt=c2ux.\begin{split} u_t &= z_x, \\ z_t &= c^2 u_{x}. \end{split}ut​zt​​=zx​,=c2ux​.​
(12.4.6)

This second form is appealing in part because it’s equivalent to Maxwell’s equations for electromagnetism. In the Maxwell form we typically replace the velocity initial condition in (12.4.3) with a condition on zzz, which may be physically more relevant in some applications:

u(x,0)=f(x),0≤x≤1,z(x,0)=g(x),0≤x≤1.\begin{split} u(x,0) &= f(x), \qquad 0 \le x \le 1, \\ z(x,0) &= g(x), \qquad 0 \le x \le 1. \end{split}u(x,0)z(x,0)​=f(x),0≤x≤1,=g(x),0≤x≤1.​
(12.4.7)

Because waves travel in both directions, there is no preferred upwind direction. This makes a centered finite difference in space appropriate. Before application of the boundary conditions, semidiscretization of (12.4.6) leads to

[u′(t)z′(t)]=[0Dxc2Dx0][u(t)z(t)]. \begin{bmatrix} \mathbf{u}'(t) \\[2mm] \mathbf{z}'(t) \end{bmatrix} = \begin{bmatrix} \boldsymbol{0} & \mathbf{D}_x \\[2mm] c^2 \mathbf{D}_x & \boldsymbol{0} \end{bmatrix} \begin{bmatrix} \mathbf{u}(t) \\[2mm] \mathbf{z}(t) \end{bmatrix}.[u′(t)z′(t)​]=[0c2Dx​​Dx​0​][u(t)z(t)​].
(12.4.8)

The boundary conditions (12.4.2) suggest that we should remove both of the end values of u\mathbf{u}u from the discretization, but retain all of the z\mathbf{z}z values. We use w(t)\mathbf{w}(t)w(t) to denote the vector of all the unknowns in the semidiscretization. When computing w′(t)\mathbf{w}'(t)w′(t), we extract the u\mathbf{u}u and z\mathbf{z}z components, and we use dedicated functions for padding with the zero end values or chopping off the zeros as necessary.

12.4.2Variable speed¶

An interesting situation is when the wave speed ccc changes discontinuously, as when light passes from one material into another. For this we must replace the term c2c^2c2 in (12.4.8) with the matrix diag⁡(c2(x0),…,c2(xm))\operatorname{diag}\bigl(c^2(x_0),\ldots,c^2(x_m)\bigr)diag(c2(x0​),…,c2(xm​)).
